# Words

Word trainer Telegram bot in Clojure

## Installation

`git clone https://github.com/kipelovets/words-clj.git`

## Usage

`make`

## Done

* Fetch translations from a dictionary
* Buttons: start exercise, my stats
* Remove word
* Select from-to languages
